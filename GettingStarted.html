Verify that Ruby is Installed

If you are running on Mac OS X right now (which you should be) then you are in luck because Ruby should already be installed! You can verify this by opening a terminal and entering the following command: ruby -v to see the current version of Ruby. If this doesn't work, or if you are running Windows, then follow instructions here to get Ruby installed.

Get Comfortable with Interactive Ruby

Ruby ships with something called the IRB (Interactive Ruby), which allows you to write and execute Ruby code in real time! This is an easy way to try Ruby out and see what happens with instant gratification. To start the IRB, open a terminal and enter irb. This should start an interactive session with Ruby and allow you to enter Ruby code and see the result. Try typing a simple command, such as 3 * 4.

Define a Method to Greet the World

You can even define classes and methods in the IRB. Methods are defined in Ruby with def method_name, followed by the method body, and ended with end. Enter the following, line by line, into the IRB:

				
					def hi
						puts "Hello World"
					end

Now you can say "Hello World" by simply referencing this method. Go ahead, tell the world hello.

Define a Better Method

Maybe you want to be able to say hello to more specific people than just the world though? Let's just redefine our hi method to accept a name as a parameter. That's the great thing about Ruby, it's flexible enough that you can redefine methods whenever you want (you can even redefine methods from core classes, such as writing your own String.puts method).

				
					def hi(name="World")
						puts "Hello #{name.capitalize}"
					end

Now your method takes an optional parameter that will be used in the greeting. Go ahead and say hello to a friend with hi("friend") or hi "friend".
